# CFStatusBarHUD
一个简单易用的HUD指示器框架
## 加载成功提示
```objc
[CFStatusBarHUD showSuccess:@"加载成功!"];
```
## 加载失败提示
```objc
[CFStatusBarHUD showError:@"加载失败!"];
```
## 正在加载中提示
```objc
[CFStatusBarHUD showLoading:@"正在加载中..."];
```
## 普通消息提示
```objc
[CFStatusBarHUD showMessage:@"普通消息" image:[UIImage imageNamed:@"check"]];
```
